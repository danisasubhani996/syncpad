SyncPad is a real-time collaborative document editor that allows multiple users to edit documents simultaneously with live synchronization.

[SyncPad Live App](https://syncpad-iokv7ioky-danisasubhani996s-projects.vercel.app?utm_source=chatgpt.com)

## 📂 GitHub Repository

[SyncPad GitHub Repository](https://github.com/danisasubhani996/syncpad.git?utm_source=chatgpt.com)

---

## ✨ Features

* Real-time collaborative editing
* Rich text editor
* Live user count
* Auto-save functionality
* Shareable document links
* Socket.IO based synchronization
* Responsive modern UI
* MongoDB cloud database integration

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/danisasubhani996/syncpad.git
cd syncpad
```

---

### 2️⃣ Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

### 3️⃣ Backend Setup

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `server` folder:

```env
MONGO_URI=your_mongodb_connection_string
```

Create a `.env` file inside the `client` folder:

```env
VITE_API_URL=http://localhost:5000
```
